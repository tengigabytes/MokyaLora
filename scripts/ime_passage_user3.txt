嵌入式系統開發筆記
最近在學習如何使用一款開源的微控制器，型號是常見的 32 位元 ARM Cortex-M 系列。開發環境設定花了一些時間，主要是要把 toolchain 和 IDE 整合起來。我選擇了 VS Code 搭配開源外掛，這樣比較方便管理多個專案。
首先做了一個基本的 LED 閃爍範例，確認硬體和燒錄流程都正常之後，才開始寫真正的應用程式。這次練習的目標是讀取一顆 I2C 介面的溫濕度感測器，把資料透過 USB CDC 傳到電腦上做記錄。看似簡單，實際上要處理的細節不少，包含時序、錯誤處理、緩衝區管理等等。
寫程式的時候習慣每完成一個小功能就 commit 一次，這樣即使後面出錯也容易回溯。版本控制是嵌入式開發的好朋友，特別是當你同時調整硬體和軟體的時候。
